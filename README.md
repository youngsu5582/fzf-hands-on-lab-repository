## 실습 목차

실습은 타이핑을 따라하는 것 위주로 진행됩니다.

1. 쉘 스크립트 기본 문법 살펴보기
2. 쉘 스크립트 함수에서 CLI 사용하기
3. fzf 사용해보기
4. fzf 를 활용한 git stash / branch 명령어 만들기

## 실습 가이드

1. step0 을 참고해서 fzf 를 설치해주세요.
2. 터미널을 키고, 아래 명령어를 실행해주세요.

```shell
chmod +x ./setup-branch.sh && ./setup-branch.sh
chmod +x ./setup-stash.sh && ./setup-stash.sh
```

3. step 디렉토리로 이동해주세요.

```shell
cd ./step
```

4. 발표에 따라, 실습을 따라서 진행합니다.
